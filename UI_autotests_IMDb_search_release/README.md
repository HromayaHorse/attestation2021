Инструкция для запуска эмуляции мобильного девайса:

В wdio.conf.js необходимо раскоментить блок кода 

'goog:chromeOptions': {
            // Здесь можно указать девайс
            mobileEmulation: { deviceName: 'iPad' },
        }
